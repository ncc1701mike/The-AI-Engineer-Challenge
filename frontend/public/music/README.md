# Background Music

This directory is for storing background music files.

## Recommended Music

For a Minecraft-like soothing soundscape, you can download royalty-free ambient tracks from:

1. **Pixabay** - Search for "minecraft" or "ambient" music:
   - https://pixabay.com/music/search/minecraft/
   - Recommended: "Minecraft Song" by SpadeStars (ID: 126368)

2. **Free Music Archive** - Ambient/Calm categories:
   - https://freemusicarchive.org/genre/Ambient

3. **Incompetech** - Kevin MacLeod's royalty-free music:
   - https://incompetech.com/music/royalty-free/

## File Setup

1. Download your preferred ambient track
2. Convert to both MP3 and OGG formats for browser compatibility
3. Name the files:
   - `ambient-background.mp3`
   - `ambient-background.ogg`
4. Place both files in this directory

The component will automatically use the local files if available, otherwise it will fall back to an external source.
